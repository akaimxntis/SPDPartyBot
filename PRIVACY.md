# SPD Party Bot Privacy Policy

Last updated: 2026-05-08

This Privacy Policy explains what data **SPD Party Bot** ("the bot") stores and how it is used.

## 1. Data collected by the bot

The bot stores only the information required for its features to work.

Depending on server configuration and usage, the bot may store:

- Discord server IDs
- Channel IDs
- Role IDs
- User IDs of party hosts and participants
- Party titles
- Party descriptions
- Party times
- Party image URLs
- Party status, such as open or closed
- Queue and participant lists
- Server-specific settings, such as language, timezone, embed color, reminder settings and auto-close settings

## 2. Data not collected

The bot does not collect or request:

- Discord account passwords
- Discord user tokens
- Personal login credentials
- Payment information
- Private messages outside of bot interactions
- Message content unrelated to the bot's party features

The bot uses slash commands, buttons and forms. It does not need Message Content Intent for its main features.

## 3. How data is used

Stored data is used to provide bot features, including creating party invites, showing participants, managing slots and queues, updating party embeds, saving server settings, sending reminders, automatically closing parties and sending logs to the configured log channel.

## 4. Data storage

Data may be stored in JSON files or another storage backend depending on the hosting setup.

If hosted on Railway or another provider, data may be stored in a persistent volume configured by the bot owner.

## 5. Data sharing

The bot does not sell user data.

The bot does not intentionally share stored data with advertisers or unrelated third parties.

Data may be visible inside Discord servers depending on how server staff configure the bot. For example, party embeds may show participant mentions or host mentions.

## 6. Data removal

Server owners or authorized staff can remove closed party data using the bot's cleanup command.

If you need data removed, contact the bot owner through the GitHub repository:

https://github.com/akaimxntis/SPDPartyBot

## 7. Server staff responsibility

Server owners and staff are responsible for configuring the bot correctly, moderating user-generated party content, removing inappropriate content and managing channel and role permissions.

## 8. Children's privacy

The bot is intended for use within Discord communities and follows Discord's platform rules. The bot should not be used to collect personal information from children.

## 9. Changes to this policy

This policy may be updated from time to time. Continued use of the bot after changes means you accept the updated policy.

## 10. Contact

For questions or privacy requests, use the GitHub repository:

https://github.com/akaimxntis/SPDPartyBot
